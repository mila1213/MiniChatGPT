```md id="r8k3sd"
# MiniChatGPT HF

MiniChatGPT es un chat simple basado en inteligencia artificial que usa Node.js en el backend y la API de Hugging Face para generar respuestas.

El proyecto permite interactuar con diferentes modelos de IA desde una interfaz web hecha con HTML, CSS y JavaScript.

## Funciones

- Chat con inteligencia artificial usando Hugging Face
- Selección de modelos (Llama, Mistral, Qwen, etc.)
- Diferentes modos de ayuda (JavaScript, Web, Inglés, Debug)
- Interfaz web simple y responsive
- Backend con Node.js y Express

## Tecnologías

- Node.js
- Express
- Hugging Face API
- HTML, CSS, JavaScript

## Instalación

```

git clone [https://github.com/mila1213/MiniChatGPT.git](https://github.com/mila1213/MiniChatGPT.git)
cd MiniChatGPT
npm install

```

## Configuración

Crear archivo `.env` en backend:

```

HF_TOKEN=tu_token_de_huggingface
PORT=3000

```

## Ejecución

```

node server.js

```

## Uso

Abrir en el navegador:

```

[http://localhost:3000](http://localhost:3000)

```

## Notas

- El token de Hugging Face es necesario para que funcione el chat
- No subir el archivo `.env` a GitHub
- Proyecto con fines educativos
```
